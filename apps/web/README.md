# Web App

## Features

- **Dark Mode**: Toggle between light and dark themes using the switch in the navigation bar. The app remembers your preference for future visits.

## How to Use

1. Open the web app.
2. Use the switch in the top navigation bar to toggle between light and dark mode.
3. Your preference will be saved and applied automatically on your next visit.

## Known Issues

- Ensure your browser supports `localStorage` and `matchMedia` for optimal performance.

## Development

To contribute to this project, please follow the standard Git workflow and ensure all changes are tested across different browsers.
